Managing Complex Technical Projects with Stride: A Real-World Case Study

January 9, 2026 tools productivity 9 min read

The Challenge: Managing Unknown Complexity

I’m building an IoT espresso machine controller called “Silvia” using Elixir and Nerves on a Raspberry Pi Zero W. When I decided to implement a TSIC306 temperature sensor, I knew it would be complex, but I didn’t know how complex or what problems I’d encounter.

This is the reality of technical work: you can’t fully plan what you don’t yet understand. Traditional project management tools want you to define all tasks upfront, estimate completion times, and follow a predetermined path. But embedded systems work—especially when you’re working with unfamiliar hardware protocols—is inherently discovery-driven.
